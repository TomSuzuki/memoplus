# memoplus
### メモ帳＋ ver 1.30

　複数ページを切り替えて使用することができるテキストです。  
　複数ページあるテキストエディタ的なツールがあるとバグの管理やアイディアの管理をしやすいだろうなと思って作りました。

## ファイルの構造  
　main.hsp    ;メインのファイル  
　　lib.hsp   ;内部処理関連の関数など  
　　lib2.hsp  ;大型の関数など  
　　lib3.hsp  ;新しく追加した関数（テスト用など）  
　　lib4.hsp  ;文字列系の関数
　config.csv  ;設定の保存用ファイル
